# PropLens Admin Panel

instructions to install, build, run, and run tests.

## Install
```bash
npm install
```

## Run (Dev)
at http://localhost:3000
```bash
npm run dev
```

## Build
Creates a production build
```bash
npm run build
```

## Run (Production)
Serves the production build
```bash
npm run start
```

## Tests (Cypress E2E)
- Headless run
```bash
npm run test:e2e
```

- Browser with head
```bash
npm run cypress:open
```
